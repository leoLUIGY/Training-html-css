Sobre mim

Iniciei em tecnologia aprendendo lógica no Scratch e evoluí para Python, Java e C#. Sou graduado em Jogos Digitais pela FATEC, onde desenvolvi projetos multidisciplinares, pratiquei pitching, documentação técnica e apresentações.

Minha primeira experiência profissional foi como jovem aprendiz administrativo, e depois atuei como monitor acadêmico, auxiliando em projetos finais. Segui com desenvolvimento de jogos independentes (Play Store, Itch.io) até ingressar como desenvolvedor Unity na 4Dmais, onde trabalhei em VR/AR (migração de Vuforia para AR Foundation, treinamentos com Oculus XR), aplicando OOP, SOLID, Clean Code e Design Patterns, além de ter contato com JavaScript, React e APIs.

Possuo conhecimentos sólidos em Programação Orientada a Objetos (POO), versionamento com Git e práticas de desenvolvimento ágil. Atualmente, estou em transição para back-end, estudando .NET, C#, SQL, Docker, REST APIs e Laravel, com foco em arquitetura de software e boas práticas. Minha principal habilidade é aprender rápido e aplicar novos conhecimentos em soluções que geram valor real para usuários e empresas.
